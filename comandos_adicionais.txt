==================== Comandos úteis Java ====================

Maiúsculo
- nome.toUpperCase();

Minúsculo
- nome.toLowerCase();


Remover espaços nas bordas
- texto.trim();


Verificar se string está vazia
- texto.isEmpty();

Verificar se string está vazia ou só tem espaços
- texto == null || texto.isBlank();


Verificar se é letra
- Character.isLetter(c);

Verificar se é número
- Character.isDigit(c);


Checar quantidade de itens em lista
- lista.size();


Valor padrão se for null
- Objects.requireNonNullElse(texto, "default");

Exemplo:
String nome = Objects.requireNonNullElse(scanner.nextLine(), "sem nome");


Quantidade de caracteres
- texto.length();


AND lógico
- &&


NOT lógico
- !


OR lógico
- ||


Comparar textos
- texto.equals("Davi");

Ignorando maiúsculas/minúsculas
- texto.equalsIgnoreCase("davi");


Pegar caractere específico
- texto.charAt(0);


----- LISTAS -----

Adicionar item em lista
- lista.add(item);


Remover item da lista
- lista.remove(item);


Verificar se lista contém item
- lista.contains(item);


Criar lista vazia
- List<String> lista = new ArrayList<>();
